DCTTabBar Class Reference

Inherits from UIView
Declared in DCTTabBar.h

Overview

DCTTabBar is displayed by a DCTTabBarController and can be subclasses to give a custom look, as the default look is a little plain.

Tasks

  • selectedItem The currently selected item. Will exist in the items array.
  • items An array of UITabBarItems that is represented by the tab bar.
  • itemHitAreas An array of hit areas. These should correspond to the UITabBarItems in the items array.
  • delegate The delegate for the tab bar.
